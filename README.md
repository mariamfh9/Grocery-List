
# Grocery List

### Overview

An App that keeps track of your grocery list!


### Installation Instructions

Run bundle install like so: 

```
$ bundle install 

```

### Usage

Type the following to startup the rails server: 

```
$ rails s
```
and open up index.html in the browser to view content.
